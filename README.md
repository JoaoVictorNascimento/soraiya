# Soraiya Command Line Sound

Sorrraiya

## Compatibility

- Linux
- Windows
- Mac

## Installing
Install globally:

    npm install -g soraiya

## Running
Run command:

    soraiya

Or import in NodeJS script:

    import soraiya from 'soraiya';

    soraiya();
    
## Running with [npx](https://www.npmjs.com/package/npx)
Run command:

    npx soraiya
